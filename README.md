# Country

A simple drupal 8 module that automatically creates `country` vocabulary and prepopulates the terms with all available countries.

# Description

Fields:
- Name (text)
- Official name (text)
- Code2l (text)
- Code3l (text)
- Flag (image)

# Usage

- Download the module.
- Install the module (`drush en country -y`)
- Run (`drush updatedb -y`)
